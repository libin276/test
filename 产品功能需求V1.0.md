# 《能源管家》产品需求说明书V1.0

> 确认日期：2025年6月20日  
> 版本号：V1.0

## 一、产品概述

能源管家是一套基于物联网和大数据的能耗监测与管理平台，实现对水、电、气、热等多能源的实时采集、分析、优化和可视化，助力企业降本增效与绿色发展。

## 二、详细功能需求

### 1. 项目用能

#### 1.1 用能概况

- 页面分为左侧导航栏、顶部项目切换栏、主内容区三部分。
- 分类能耗统计卡片：
  - 展示电、水、碳排放三类能耗，分别显示今日、本月、本年数据。
  - 每项能耗卡片包含：本期数值、环比/同比百分比、上期（昨日/上月/去年同期）数值。
  - 数值单位：电（kWh）、水（t）、碳排放（kg）。
  - 百分比为绿色向下箭头表示下降，红色向上箭头表示上升。
- 建筑信息卡片：
  - 展示建筑面积（㎡）、计量点数、上月单位面积能耗（kWh/㎡）、近12月单位面积能耗（kWh/㎡）。
  - 各项数据有信息提示说明。
- 分类能耗趋势：
  - 下拉可选能耗类型（电/水），默认电。
  - 展示当日逐时趋势曲线（今日/昨日对比），横轴为小时，纵轴为能耗数值。
  - 展示本月逐日趋势曲线（本月/上月对比），横轴为日期，纵轴为能耗数值。
  - 支持图例切换显示。
- 交互说明：
  - 所有卡片和图表均为只读展示，部分字段带有信息提示（悬浮说明）。
  - 能耗类型切换、日期切换、项目切换均为即时刷新。
- 数据说明：
  - 能耗数据来源于系统统计，碳排放按折标系数自动换算。
  - 单位面积能耗=总能耗/建筑面积。
  - 所有统计均注明统计口径和时间范围。

**交互流程：**
1. 用户进入页面，默认展示当前项目、今日/本月/本年能耗数据及趋势。
2. 用户可通过顶部下拉切换项目，页面数据即时刷新。
3. 分类能耗趋势区，用户可下拉切换能耗类型（电/水），趋势图随之切换。
4. 趋势图支持切换今日/昨日、本月/上月对比，图例可单独显示/隐藏。
5. 鼠标悬浮于字段或图表节点，显示详细数值及说明。
6. 若无数据，卡片和图表显示“暂无数据”提示。

**数据口径与公式：**
- 今日/昨日/本月/上月/本年/去年：均以自然日/月/年为统计周期。
- 环比/同比百分比=（本期-上期）/上期×100%，如上期为0则显示“--”。
- 碳排放=电量×电折标系数+水量×水折标系数。
- 单位面积能耗=总能耗/建筑面积。

**异常与权限处理：**
- 无权限项目不在下拉列表中显示。
- 数据加载失败时，显示“数据加载失败，请重试”。
- 所有只读卡片和图表不可编辑。
- 仅有查看权限的用户可见本页面。

**导出规范：**
- 支持导出趋势图数据为Excel，字段包含：日期、能耗类型、数值、单位、统计口径、导出时间。
- 文件命名：用能概况趋势_导出_yyyyMMdd_HHmmss.xlsx

#### 1.2 用能统计

- 统计维度：年、月、日，支持按分类能耗类型筛选。
- 展示内容：趋势图、明细表，明细可导出Excel。
- 单位：用电kWh，用水吨，用气m³。
- 支持批量导出、分页展示。

**字段与数据说明：**

| 字段         | 说明             | 数据类型 | 单位   | 取值/格式 | 是否可为空 | 默认值 | 备注 |
|--------------|------------------|----------|--------|-----------|------------|--------|------|
| 分类能耗类型 | 能耗分类         | string   |        | 电/水/气  | 否         | 电     |      |
| 时间维度     | 统计周期         | string   |        | 年/月/日  | 否         | 日     |      |
| 用能数值     | 统计周期能耗     | number   | kWh/t/m³ | ≥0      | 否         | 0      |      |
| 明细表字段   | 详见导出Excel    | -        | -      | -         | -          | -      |      |

**交互流程：**
1. 用户选择能耗类型、时间维度，页面展示对应趋势图和明细表。
2. 明细表支持分页浏览和Excel导出。
3. 趋势图支持切换不同能耗类型和时间维度。
4. 若无数据，显示“暂无数据”提示。

**导出规范：**
- 导出文件包含所有筛选条件、字段、单位、导出时间。
- 文件命名：用能统计_导出_yyyyMMdd_HHmmss.xlsx

#### 1.3 同比分析

- 统计维度：年、月、日，按分类能耗类型筛选。
- 展示内容：本期、同期能耗柱状图及明细表。
- 明细表字段：本期能耗、同期能耗、增加值、同比（(本期-同期)/本期）。
- 支持导出Excel，字段包含：本期能耗、同期能耗、增加值、同比。

**字段与数据说明：**

| 字段         | 说明             | 数据类型 | 单位   | 取值/格式 | 是否可为空 | 默认值 | 备注 |
|--------------|------------------|----------|--------|-----------|------------|--------|------|
| 分类能耗类型 | 能耗分类         | string   |        | 电/水/气  | 否         | 电     |      |
| 时间维度     | 统计周期         | string   |        | 年/月/日  | 否         | 日     |      |
| 本期能耗     | 当前周期能耗     | number   | kWh/t/m³ | ≥0      | 否         | 0      |      |
| 同期能耗     | 上年/上月/昨日能耗 | number   | kWh/t/m³ | ≥0     | 否         | 0      |      |
| 增加值       | 本期-同期         | number   | kWh/t/m³ | 可正可负 | 否         | 0      |      |
| 同比         | (本期-同期)/同期  | number   | %      | 正负数    | 否         | 0      |      |

**交互流程：**
1. 用户选择能耗类型、时间维度，页面展示本期与同期对比柱状图和明细表。
2. 明细表支持分页浏览和Excel导出。
3. 若无数据，显示“暂无数据”提示。

**导出规范：**
- 导出文件包含所有筛选条件、字段、单位、导出时间。
- 文件命名：同比分析_导出_yyyyMMdd_HHmmss.xlsx

#### 1.4 复费率报表

- 支持按年、月、日筛选，支路支持全选、多选、级联选择。
- 超过50条支路分页查询。
- 展示内容：分、峰、平、谷、深谷各时段电量、电费及合计。
- 支持批量导出，导出命名规范：复费率统计+时间范围。

**字段与数据说明：**

| 字段         | 说明             | 数据类型 | 单位   | 取值/格式 | 是否可为空 | 默认值 | 备注 |
|--------------|------------------|----------|--------|-----------|------------|--------|------|
| 支路         | 支路名称         | string   |        |           | 否         | -      |      |
| 时间维度     | 统计周期         | string   |        | 年/月/日  | 否         | 日     |      |
| 尖/峰/平/谷/深谷电量 | 各时段电量 | number   | kWh    | ≥0        | 否         | 0      |      |
| 尖/峰/平/谷/深谷电费 | 各时段电费 | number   | 元     | ≥0        | 否         | 0      |      |
| 合计         | 总电量/总电费    | number   | kWh/元 | ≥0        | 否         | 0      |      |

**交互流程：**
1. 用户选择时间维度、支路，页面展示各时段电量、电费明细。
2. 支持多选、全选、级联选择支路，超50条分页。
3. 明细表支持分页浏览和Excel导出。
4. 若无数据，显示“暂无数据”提示。

**导出规范：**
- 导出文件包含所有筛选条件、字段、单位、导出时间。
- 文件命名：复费率报表_导出_yyyyMMdd_HHmmss.xlsx

#### 1.5 能源流向图

- 展示项目各级支路能耗流向，支持按分类能耗切换。
- 支持自动生成（按支路层级）和手动配置（自定义层级与公式）。
- 展示设备名称、编码、回路名称、上级回路、节点属性等。
- 支持导出。

**字段与数据说明：**

| 字段         | 说明             | 数据类型 | 单位   | 取值/格式 | 是否可为空 | 默认值 | 备注 |
|--------------|------------------|----------|--------|-----------|------------|--------|------|
| 能耗分类     | 分类名称         | string   |        | 电/水/气  | 否         | 电     |      |
| 层级         | 支路层级         | string   |        | 一级/二级/三级等 | 否   | 一级   |      |
| 设备名称     | 设备名称         | string   |        |           | 否         | -      |      |
| 设备编码     | 设备编码         | string   |        |           | 否         | -      |      |
| 回路名称     | 回路名称         | string   |        |           | 否         | -      |      |
| 上级回路     | 上级回路名称     | string   |        |           | 否         | -      |      |
| 节点属性     | 节点属性         | string   |        | 一级/二级/三级等 | 否   | 一级   |      |

**交互流程：**
1. 用户选择能耗分类，页面展示对应能源流向图。

3. 支持导出流向图数据。
4. 若无数据，显示“暂无数据”提示。

**导出规范：**
- 导出文件包含所有筛选条件、字段、单位、导出时间。
